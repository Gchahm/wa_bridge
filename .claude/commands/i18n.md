Ensure all user-facing text is translatable using Lingui.

## Patterns

### JSX Content — `<Trans>`

```tsx
import { Trans } from '@lingui/react/macro'

<Trans>Save Changes</Trans>
<Trans>Welcome, {userName}</Trans>
<Button><Trans>Submit</Trans></Button>
```

### String Values — `t` template

```tsx
import { useLingui } from '@lingui/react/macro'

const { t } = useLingui()

<Input placeholder={t`Enter name`} />
const columns = useMemo(() => [
  { header: t`Name`, accessorKey: 'name' },
], [t])  // Include t in dependencies!
```

### When to Use Which

| Context | Pattern |
|---------|---------|
| Button/label/heading text | `<Trans>` |
| Input placeholder | `t` |
| Error messages | `t` |
| Table headers | `t` |
| aria-label | `t` |

## Rules

1. **Import from macro**: `@lingui/react/macro` (NOT `@lingui/react`)
2. **Include `t` in useMemo deps**: `useMemo(() => [...], [t])`
3. **No string concatenation**: Use `<Trans>Hello {name}</Trans>` not `<Trans>Hello</Trans> + name`
4. **Don't translate technical content**: DB field names, API values, universal abbreviations
